Блок PHP кода в Blade в Laravel
В некоторых случаях бывает полезно встроить PHP код в представления. Вы можете использовать Blade 
директиву @php для выполнения блока чистого PHP в вашем шаблоне:

@php
	//
@endphp
Давайте выполним какой-нибудь код:

@php
	echo 123;
@endphp
