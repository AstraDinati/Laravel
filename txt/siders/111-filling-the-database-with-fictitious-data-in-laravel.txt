Наполнение БД фиктивными данными в Laravel
В Laravel можно заполнять таблицы БД фиктивными данными. Это делается с помощью специальных 
классов сидеров. Эти классы располагаются в папке database/seeders.

По умолчанию там уже есть класс DatabaseSeeder, представляющий собой заготовку, которой можно 
пользоваться. У этого класса есть метод run, в котором мы будем прописывать команды на заполнение 
таблиц базы данных:

<?php
	class DatabaseSeeder extends Seeder
	{
	    public function run()
	    {
	        // команды
	    }
	}
?>
