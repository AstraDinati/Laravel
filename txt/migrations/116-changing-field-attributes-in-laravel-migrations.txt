Изменение атрибутов полей в миграциях Laravel
Можно изменять тип данных существующего поля. Это делается с помощью метода change. Давайте 
посмотрим на пример.

Пусть в таблице с постами мы задали поле title размером 50 символов. Давайте увеличим это поле до 
100 символов:

<?php
	public function up()
	{
		Schema::table('posts', function (Blueprint $table) {
			$table->string('title', 100)->change();
		});
	}
?>
