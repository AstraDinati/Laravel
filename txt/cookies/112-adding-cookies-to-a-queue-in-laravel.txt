Добавление куки в очередь в Laravel
Иногда может быть не очень удобно цеплять куку к объекту ответа. Ведь кука может устанавливаться в одном месте кода, а объект ответа возвращается в 
другом.

В этом случае куку можно поставить в очедедь. Такие куки затем автоматически будут прицеплены к объекту ответа. Это делается с помощью специального 
статического метода queue класса Cookie:

<?php
	use Cookie;
	
	class PostController extends Controller
	{
		public function set()
		{
			Cookie::queue('name', 'value', 10);
			return response('test');
		}
	}
?>
