Получение значения одного столбца через QB в Laravel
Можно извлечь значение одной колонки определенного ряда. Для этого используется метод value. В следующем примере
 наш запрос найдет первую запись, подпадающую под условие и возьмет из нее значение поля title:

<?php
	$title = DB::table('posts')->where('id', '1')->value('title');
	echo $title;
?>
