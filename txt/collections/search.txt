search()
Метод search ищет в коллекции указанное значение и возвращает его ключ, если он найден. Если элемент не 
найден, возвращается false:

$collection = collect([2, 4, 6, 8]);

$collection->search(4);

// 1
Поиск выполняется с использованием «гибкого» сравнения, то есть строка с целым значением будет считаться 
равной целому числу того же значения. Чтобы использовать «жесткое» сравнение, передайте true в качестве 
второго аргумента метода:

collect([2, 4, 6, 8])->search('4', $strict = true);

// false
В качестве альтернативы вы можете передать собственное замыкание для поиска первого элемента, который 
проходит указанный тест на истинность:

collect([2, 4, 6, 8])->search(function ($item, $key) {
    return $item > 5;
});

// 2
