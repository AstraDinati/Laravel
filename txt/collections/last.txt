last()
Метод last возвращает последний элемент в коллекции, который проходит указанную проверку истинности:

collect([1, 2, 3, 4])->last(function ($value, $key) {
    return $value < 3;
});

// 2
Вы также можете вызвать метод last без аргументов, чтобы получить последний элемент коллекции. Если 
коллекция пуста, возвращается null:

collect([1, 2, 3, 4])->last();

// 4
