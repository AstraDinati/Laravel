Работа с сессиями через глобальную функцию в Laravel
Для работы с сессиями можно также использовать глобальную функцию session. Давайте посмотрим, как с ней работать.

Сохраним данные в сессию:

<?php
	session(['key' => 'value']);
?>
Получим данные из сессии:

<?php
	$value = session('key');
?>
При получении укажем значение по умолчанию:

<?php
	$value = session('key', 'default');
?>
